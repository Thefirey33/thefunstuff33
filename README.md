# Thefirey33's Fun Stuff.

A website where I upload whatever game, activity I can make in web development.

## Requirements

- npm

...and that's pretty much it.


## Running & Building for Production

To run this website, use:

```shell
npm run dev
```

To build this website for production, use:

```shell
npm run dev
```

Then, after it's built, test it with:

```shell
npm run preview
```
